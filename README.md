# 🌐 SocialNest – Social Media Platform

## 📌 Overview

**SocialNest** is a full-stack social media platform developed as part of my **CodeAlpha Full Stack Development Internship**. It provides essential social networking features such as user authentication, profile management, post sharing, likes, comments, and a follow system. The project focuses on building a secure, responsive, and user-friendly social media application using modern web technologies.

---

## ✨ Features

* 🔐 User Registration & Login (JWT Authentication)
* 👤 Personalized User Profiles
* 📝 Create, Edit & Delete Posts
* ❤️ Like & Unlike Posts
* 💬 Comment on Posts
* 👥 Follow & Unfollow Users
* 🔍 Search Users
* 📱 Responsive UI for Desktop & Mobile
* 🔒 Secure REST APIs
* 🗄 MongoDB Database Integration

## 🚀 Getting Started

### Prerequisites

* Node.js
* npm
* MongoDB (Local or MongoDB Atlas)

### 1. Clone Repository

```bash
git clone https://github.com/YOUR_GITHUB_USERNAME/SocialNest.git
cd SocialNest
```

### 2. Backend Setup

```bash
cd backend
npm install
```

Create a `.env` file inside the **backend** folder.

```env
MONGO_URI=your_mongodb_connection_string
PORT=5000
JWT_SECRET=your_secret_key
```

Run the backend server:

```bash
npm run dev
```

or

```bash
npm start
```

---

### 3. Frontend Setup

Open the **frontend** folder using **Live Server** or any static web server.

Backend URL:

```
http://localhost:5000
```

Frontend URL:

```
http://127.0.0.1:5500
```
